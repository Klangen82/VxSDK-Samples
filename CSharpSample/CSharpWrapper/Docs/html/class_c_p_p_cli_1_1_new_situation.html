If less than 0, a generated event must be manually acknowledged. If 0, a generated event is set to AutoAcked immediately.

System:: String^ CPPCli::NewSituation::Type
getset

Gets or sets situation type.

The type must be of the form external/<company>/<event> where <company> and <event> are strings no greater than 64 characters each; forward slashes are not allowed. These strings describe the <company> that manufactured the device that was the source of the <event>.
